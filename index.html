Mi prueba de git

Esto es un ejercicio de prueba de funcionamiento del sistema de repositorio git

Ejercicios propuestos

  1. Crea un proyecto local, con un archivo html y otro css.
  2. Haz un seguimiento en git
  3. Crea un commit inicial
  4. Sube el proyecto a github
  5. Añade al menos un compañero al proyecto
  6. Modifica el html para añadir al menos 6 elementos
